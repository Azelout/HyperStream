# Hyperstream Frontend

This is the new specific frontend for Hyperstream, built with **Vite + React + Tailwind CSS**.

## Features

- **Hyperspeed UI**: Three.js background and Glassmorphism design.
- **Role-Based**: Employer and Employee dashboards.
- **Performance**: High-speed SPA architecture.
